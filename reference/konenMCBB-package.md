# konenMCBB: College Basketball Data Digestion for R

A unified interface for pulling, parsing, and storing college basketball
data from multiple sources: Bart Torvik (barttorvik.com) for advanced
team and player efficiency metrics, ESPN for live and historical game
data (box scores, play-by-play, betting lines, win probability), Sports
Reference for rosters and player game logs, NCAA.com for play-by-play,
and Covers.com for injury data. Includes a local flat-file ESPN database
pipeline and utilities for loading data into PostgreSQL.
