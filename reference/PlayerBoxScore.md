# Get player box score game log

Retrieves a player's season game log from Sports Reference.

## Usage

``` r
PlayerBoxScore(name, season, team)
```

## Arguments

- name:

  Player name (e.g., "Zach Edey")

- season:

  Season year (numeric, e.g., 2024)

- team:

  Team name to match

## Value

A data.frame of game logs
